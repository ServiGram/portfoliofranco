Portafolio

Multigiros

Multigiros es una aplicación desarrollada por encargo de un cliente para gestionar el control de usuarios y transacciones de giros internacionales. La app fue creada para dispositivos Android utilizando Java, con Firebase como sistema de gestión de bases de datos. Entre sus principales funcionalidades, destaca el manejo de distintos niveles de inicio de sesión y la administración eficiente tanto de los datos de los clientes como de las cuentas bancarias a las que se realizan los giros o remesas de forma regular.

VidriAluminios

VidriAluminios es una aplicación desarrollada específicamente para gestionar y optimizar las operaciones de mi empresa. Creada en Kotlin para Android, esta herramienta permite controlar de manera eficiente las entradas y salidas de materiales, gestionar el corte de los mismos, así como administrar la cartera de clientes y las cotizaciones. La aplicación fue diseñada con el objetivo de agilizar y simplificar diversos procesos, facilitando el trabajo diario de los empleados y mejorando la productividad en general.

AutoPrestamo

AutoPrestamo es una aplicación web desarrollada en Angular para una empresa, que ofrece tanto una plataforma interactiva para los clientes como un dashboard exclusivo para administradores. A través de este panel, los administradores pueden gestionar y controlar los préstamos otorgados, con la capacidad de visualizar las cuotas generadas diariamente. Además, el sistema integra diversas funcionalidades adicionales para facilitar la administración eficiente de los préstamos. La conexión a la base de datos se gestiona mediante Firebase, asegurando un manejo rápido y seguro de la información. Ir a la página AutoPrestamo
